Save your Zoom
Live Transcript
in one command.

Zoom lets you see and select Live Transcript text — but not copy it. This tool uses your OS's built-in Accessibility API to grab the full transcript and save it to your Desktop.

Terminal — one-liner
curl -sSL https://raw.githubusercontent.com/Xiaoniu86/zoom-transcript-grabber/main/mac/grab.sh | bash

First run requires granting Accessibility permission — the script will walk you through it.

PowerShell — one-liner
irm https://raw.githubusercontent.com/Xiaoniu86/zoom-transcript-grabber/main/windows/grab.ps1 | iex

No extra permissions needed on Windows.

How to use

Three steps

1

Open the Transcript panel in Zoom

During a meeting, click CC / Captions in the bottom toolbar → View Full Transcript

2

Run the command above

Mac: open Terminal. Windows: open PowerShell. Paste the command and press Enter.

3

Find the file on your Desktop

A file named zoom_transcript_TIMESTAMP.txt will appear on your Desktop with the full content.

Risks & Permissions

Please read before running

⚠️ Mac: Accessibility permission

This tool requires you to grant Terminal the Accessibility permission. Once granted, any script running in Terminal can read text from other app windows.

  • Only run scripts you trust
  • Revoke the permission after you're done (the script will prompt you)
  • Manual revoke: System Settings → Privacy & Security → Accessibility → uncheck Terminal

⚠️ Meeting content & privacy

The transcript may contain other participants' speech. Please:

  • Follow your organization's policies on meeting recordings
  • Do not share others' words without consent

✅ What this tool will never do

  • Upload or transmit any data over the network
  • Install software or modify system files
  • Read content from any window other than Zoom's Transcript panel
  • Leave any persistent process on your computer
How it works

Why can it bypass Zoom's copy block?

🔒

How Zoom blocks copying

Zoom disables right-click menus and intercepts Ctrl+C. But the text is still rendered on screen — the OS knows it's there.

♿

Accessibility API

macOS and Windows provide a system-level API so screen readers can read any app's UI text. Zoom cannot hide text from the OS — otherwise blind users couldn't use it.

📖

Read-only by design

The Accessibility API is read-only. It can read text, but it cannot click buttons, modify content, or perform actions on your behalf.

💾

Saved locally

The text is written directly to a local .txt file. Nothing is sent over the network at any point.
